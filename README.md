# LIO-MOVIE-RECOMMANTATION-CHATBOX
Lio Movie Chatbot: Fullscreen Tkinter app (dark blue theme, cyan buttons) for movie recommendations.  Flow: Name → Mood (Good/Okay/Bad) → Age group → Language (English/Hindi) → 2 personality Qs tally 8 genres (A-Romance to H-Indian).  Output: Top genre's age-appropriate YouTube trailers , examples, Restart/Exit. Launch: start_main_chatbot().
